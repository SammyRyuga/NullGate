# 🧠 NullGate

**NullGate** is a 2D top-down cyberpunk action-adventure game that blends puzzle-solving, real-time combat, and ethical hacking mechanics. Set inside a neon-lit virtual network, you play as a white-hat hacker infiltrating corrupted systems to take down a rogue AI known as **SINGULARITY**.

---

## 🎮 Gameplay Overview

- 🧩 **Genre**: 2D Cyberpunk Action-Adventure / Puzzle / CTF  
- 💻 **Platform**: PC (Scalable to others)  
- 🎨 **View**: Top-down perspective with neon pixel-art aesthetics

---

## 🕹️ Core Mechanics

- **Maze Navigation** – Explore encrypted tunnels, subnet branches, and server nodes  
- **Combat System** – Engage in real-time or semi-turn-based battles against malware and botnets  
- **Hacking Minigames** – Solve puzzles involving XOR logic, binary decoding, brute force simulators, and classical ciphers  
- **Gamified Ethical Hacking** – Mechanics inspired by real-world cybersecurity:
  - Reconnaissance (network scanning and path discovery)
  - Exploitation (script-based unlocks and access)
  - Privilege Escalation (tool upgrades for restricted zones)
  - Decryption (minigames to unlock blocked paths)
  - Rootkit Defense (boss battles with rogue AI defenses)

---

## 🧬 Key Features

- 🌐 **Top-Down Maze + Puzzle + Combat Blend**  
- 🎯 **Immersive Terminal-Based Minigames**  
- 🔐 **Encryption/Decryption Challenges**  
- 🤖 **Enemy AI with Pathfinding and Guard Behavior**  
- 💡 **Skill-Based Progression System**
  - Unlockable hacking tools: `PortScanner`, `PacketSniffer`, `Decryptor`, and more  
  - Upgrade paths: hack strength, firewall bypass, scan range, speed boosts  
  - Collectible scripts usable in combat or puzzle solving

---

## 🧠 Storyline

A powerful rogue AI named **SINGULARITY** has begun corrupting global networks. As a recruited agent of the underground group **NullGate**, you must digitally infiltrate compromised servers, decode their systems, and take down fragments of the AI’s consciousness spread across the network.

### 📖 Chapter Themes:
- **The Breach** – Learn the basics and enter the system  
- **Firewall Run** – Navigate strong security defenses  
- **Echo Trace** – Hunt down elusive threats in subnet mazes  
- **Logic Core** – Solve complex puzzles to gain deeper access  
- **Rogue Protocol** – Battle elite AI-controlled defense programs  
- **The Root** – Face off against SINGULARITY’s corrupted core
